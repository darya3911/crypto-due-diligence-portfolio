# My Crypto Research Toolkit: Useful Links

Over the past couple of years, I've tried hundreds of sites and services. Most of them are garbage. But some have become absolutely essential. Here's my personal collection — the stuff I actually use.

I've grouped everything by category so it's easier to navigate. Under each tool, I wrote a short note on why I use it and when.

---

## 1. Blockchain Explorers 🔍

This is where any serious research starts. If you don't know how to use explorers, you're not an analyst — you're just someone who reads news.

**[Etherscan](https://etherscan.io)** — the main tool for Ethereum. I check transactions, look at contracts, read code. If a contract isn't verified — that's already a red flag.

**[BscScan](https://bscscan.com)** — same thing but for Binance Smart Chain. A lot of scam projects love BSC, so you can't avoid it.

**[Solscan](https://solscan.io)** — for Solana. Solana has its own logic, but the explorer works the same way.

**[Arbiscan](https://arbiscan.io)** / **[Polygonscan](https://polygonscan.com)** / **[Optimistic Etherscan](https://optimistic.etherscan.io)** — for L2 solutions. If a project is on L2, I check here.

**[OKLink](https://www.oklink.com)** — multi-chain explorer from OKX. Handy when you need quick data across different chains in one place.

**[Blockchair](https://blockchair.com)** — a search engine across several blockchains at once. Useful for cross-chain analysis.

---

## 2. On-Chain Analytics 📊

Explorers show the facts. These platforms help interpret them.

**[Nansen](https://nansen.ai)** — expensive but the best. Wallet labeling (whales, funds, exchanges) and tracking "smart money." If whales are moving into a project, Nansen will show it first.

**[Dune Analytics](https://dune.com)** — absolutely brilliant. The community creates dashboards for everything you can imagine. Want to see how many active users a project has? Go to Dune. Someone already made the chart.

**[Glassnode](https://glassnode.com)** — macro metrics for Bitcoin and Ethereum. MVRV, SOPR, realized cap. For understanding market cycles — must-have.

**[CoinMetrics](https://coinmetrics.io)** — institutional-grade data. Comparing crypto assets with traditional ones, net flows, on-chain metrics in Excel.

**[CryptoQuant](https://cryptoquant.com)** — focuses on exchange flows. I track how many coins are moving onto exchanges and off them. Before pumps, there's often an outflow from exchanges.

**[Token Terminal](https://tokenterminal.com)** — a financial lens on crypto. P/E, P/S, protocol revenue. Lets you build DCF models for DeFi projects.

**[DeFiLlama](https://defillama.com)** — the main TVL aggregator. If a project is in DeFi, this is the first place I look: how much money is locked, growing or falling, who are the competitors.

**[DappRadar](https://dappradar.com)** — for dApp analysis. Sort by users, volume, chains. Convenient for monitoring activity.

**[DeBank](https://debank.com)** — managing DeFi portfolios and checking permissions. Shows which contracts you've given access to your funds. For security — indispensable.

---

## 3. Market Data Aggregators 📈

Quick glance at the market, prices, volumes.

**[CoinMarketCap](https://coinmarketcap.com)** — the classic. Prices, market cap, volume. I also watch the "Recently Added" section — fresh projects often pop up there.

**[CoinGecko](https://coingecko.com)** — alternative to CMC. I like it for more detailed community and developer analytics.

**[TradingView](https://tradingview.com)** — the best tool for technical analysis. Charts, indicators, community ideas. I use it every single day.

**[DEX Screener](https://dexscreener.com)** — for tracking freshly launched tokens on decentralized exchanges. If a project just got listed on Uniswap, you'll see it here immediately.

**[CoinGlass](https://coinglass.com)** — futures and derivatives data. Liquidation heatmaps, funding rates, open interest. Great for understanding market sentiment.

**[LunarCrush](https://lunarcrush.com)** — social analytics. Shows which tokens are being discussed on social media and how that correlates with price. For meme coins — essential.

## 4. Tools for Researching Projects and Teams 👥

This is where I check who's behind the project and where their money comes from.

**[RootData](https://rootdata.com)** — a recent discovery. Database of projects, investors, and team. You can see who invested in whom, which funds are backing a project. For due diligence — amazing.

**[Crunchbase](https://crunchbase.com)** — for checking founders' backgrounds. If someone actually built successful startups before, it'll be here.

**[LinkedIn](https://linkedin.com)** — I always check team profiles. Are the accounts real? Is there history? Recommendations?

**[X (Twitter)](https://twitter.com)** — I follow the developers, not the influencers. Often important news appears in tweets first, then in official announcements.

**[GitHub](https://github.com)** — I look at the project's repositories. Commit activity, number of developers, code quality. Empty GitHub — red flag.

**[Dove Metrics](https://dovemetrics.com)** — tracking venture deals. Who, when, and how much invested. Useful for gauging fund interest.

**[CryptoRank.io](https://cryptorank.io)** — aggregator of ICO/IDO data, fundraising, and token unlock schedules.

## 5. Security and Audit 🔒

Before I trust a project, I check if others have checked it first.

**[CertiK](https://certik.com)** — top-tier auditor. If a project has an audit from CertiK, that's already a serious claim. I look at reports on their site.

**[Hacken](https://hacken.io)** — also among the leaders. Especially many audits in DeFi.

**[Cyberscope](https://cyberscope.io)** — good auditor, often work with BSC projects.

**[The De.Fi](https://de.fi)** — risk dashboard. Shows security scores for different projects.

**[RugDoc](https://rugdoc.io)** — specialize in identifying scams. Check code for hidden functions that allow developers to steal money.

**[Token Sniffer](https://tokensniffer.com)** — quick token check for basic red flags (honeypot, hidden mint, etc.).

**[ETH Gas Station](https://ethgasstation.info)** — monitoring Ethereum network fees. So you don't overpay for transactions.

## 6. Communities and News 📰

Keeping my finger on the pulse.

**[CoinDesk](https://coindesk.com)** — the main news media. No hype, just facts.

**[CoinTelegraph](https://cointelegraph.com)** — more visuals, still decent.

**[The Block Research](https://theblock.co)** — deep analytical reports. For diving deep into topics.

**[Messari](https://messari.io)** — institutional reports, project profiles, news. Expensive but worth every penny.

**[Reddit](https://reddit.com)** — subreddits like r/CryptoCurrency and r/ethdev. Sometimes insights pop up there that you won't find in mainstream media.

## 7. Event Calendars 📅

So I don't miss important stuff.

**[RootData Calendar](https://rootdata.com)** — token unlocks, updates, airdrops.

**[CoinMarketCap Calendar](https://coinmarketcap.com/calendar)** — exchange listings, project events.

**[TokenUnlocks](https://tokenunlocks.com)** — specialized service for token unlocks. I check when project tokens come out of lock-up and whether it'll put pressure on the price.

---

## How I Actually Use This

I don't have a rigid "always check everything" scheme. But usually the order goes like this:

1. **Quick glance:** CoinMarketCap / CoinGecko — what's this project?
2. **Team:** LinkedIn + RootData + Twitter — who are these people?
3. **Code & security:** Etherscan + audits (CertiK/Hacken) — what's inside?
4. **Community:** Twitter + Telegram/Discord — real people or bots?
5. **Finances:** DeFiLlama + Token Terminal — how much money and where from?
6. **Market:** DEX Screener + TradingView — how's it trading?

This isn't the ultimate truth. Sometimes the order changes. But this is the foundation.
